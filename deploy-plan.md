# План развертывания dosmundos.pe на VPS

## Цель
Настроить автоматическое развертывание проекта dosmundos.pe с GitHub репозитория на VPS сервер.

## Анализ проекта
Проект представляет собой React приложение с Vite, включающее:
- Frontend на React с множеством компонентов
- API endpoints для AssemblyAI и прокси аудио
- Supabase интеграция
- Множество плагинов и хуков
