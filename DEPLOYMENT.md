# Обновление VPS

## Ручной деплой (рекомендуется)

1. Подключитесь к вашему VPS по SSH
2. Перейдите в директорию проекта:
   ```bash
   cd /path/to/your/project
   ```
3. Запустите скрипт обновления:
   ```bash
   ./deploy-manual.sh
   ```

## Автоматический деплой (GitHub Actions)

Автоматический деплой запускается при пуше в ветку `main`, но может иметь проблемы с правами доступа.

Если автоматический скрипт не работает:

1. Подключитесь к VPS по SSH
2. Перейдите в директорию проекта
3. Выполните команды:
   ```bash
   git pull origin main
   npm install
   npm run build
   pm2 restart dosmundos-server
   ```

## Проверка обновления

После обновления проверьте, что сервер работает:
```bash
curl http://localhost:3000/api/health
```

Должно вернуть JSON с `{"status":"ok",...}`
